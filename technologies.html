Opening Trigrams Subjects Technologies Essay

This embedded chart shows the percentage of books containing the technology word.

The Trigrams table was required when searching for words above a given frequency.

Google Books Ngram Viewer tool works when searching for frequencies of words.

The tool can run case-insensitive searches so the percentage is now exact.

The tool does not allow for also combining words (e.g. "phone" & "telephone") so some math is required.

In 1800s the telegraph was on top, 1900-30 telephone, 1930-50 radio, 1950-60 television, 1960+ computer.

Creating such a chart using a history book and invention dates you would likely create a very simliar chart.

This proves the corpus can be used to understand the lives of people by analyzing the written word of people.
