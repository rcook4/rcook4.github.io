Opening Essay Trigrams Subjects Technologies

    Essay
  • My final project is a interactive slideshow.
  • The website default page is the opening scene.
  • Users can navigate to a scene using a menu link.
  • Each menu link triggers the parameter change to that chart data depth.
  • There are three data depths: Trigrams, Subjects, Technologies.
  • Each chart represents a single state.
  • Trigrams and Subjects d3 annotations are based on the block susielu (a464c24d8b42f0c4d9fafe7b48e9e60a)
  • Trigrams and Subjects d3 charts are based on the block by PatMartin (4ac4ee837b121953e272ff576b75da6c)
  • Trigrams and Subjects allow the user to blush, zoom, highlight and see tooltips
  • Technologies is an embedded chart from Google Books Ngram Viewer.
  • Users can perform adhoc analysis using Google Books Ngram Viewer.
